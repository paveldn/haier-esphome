Example of climate configuration for hOn protocol
=================================================

Configuration of your climate will depend on capabilities specific model.

Minimal configuration will look like this:

.. code-block:: yaml

    uart:
      baud_rate: 9600
      tx_pin: 17
      rx_pin: 16
    
    climate:
      - platform: haier
        protocol: hon
        name: Haier hOn Climate
